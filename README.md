# LeuwongRR API Docs

Dokumentasi Mintlify untuk LeuwongRR REST API.

## Upload ke GitHub

Upload semua file di folder ini ke repository:

```txt
https://github.com/dksan03/leuwongrr-api-docs
```

## Mintlify

Pilih repository tersebut di Mintlify, branch `main`, dan matikan toggle `Docs are in a subdirectory`.
