# LeuwongRR REST API

LeuwongRR REST API v1 digunakan untuk integrasi order, invoice, payment callback, dan akses data user secara aman.

## Base URL

```txt
https://leuwongrr.online
```

## Scope API v1

- Order API
- Invoice API
- Payment Callback API
- User API Access

API ini dibuat ringan untuk shared hosting, jadi endpoint yang dibuka hanya endpoint inti transaksi.
