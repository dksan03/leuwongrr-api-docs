# Errors

Response error memakai format JSON sederhana.

```json
{
  "success": false,
  "message": "Unauthorized API token."
}
```

## Status umum

| HTTP Status | Arti |
| --- | --- |
| 200 | Request berhasil |
| 400 | Request tidak valid |
| 401 | Token tidak valid / tidak dikirim |
| 404 | Data tidak ditemukan |
| 500 | Error server |

## Unauthorized

Pastikan header dikirim:

```http
Authorization: Bearer YOUR_TOKEN
```
